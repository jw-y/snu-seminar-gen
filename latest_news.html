SNU Data Science Seminar “Native Agent Memory in the Oracle AI Database with Open Agent Spec. ” Time : 09:00 - 10:30, Mar 11, 2026 Zoom: https://snu-ac-kr.zoom.us/j/2507466555 Abstract Agentic systems are rapidly moving beyond traditional Retrieval-Augmented Generation, yet most memory solutions remain document-centric, fragmented, and tightly coupled to specific runtimes. This talk introduces the Open Agent Specification, a framework-agnostic declarative language for defining standalone agents and structured agentic workflows, and explores how it can evolve to natively support agent memory by grounding it directly in the database. We explore how Agent Spec abstractions decouple agents from memory implementations while enabling Oracle Database to act as a converged memory substrate for agentic applications. Oracle AI DB unifies vector, graph, JSON, access controls, and in- database agent execution to power agent memory, execution-trace leaming and multi-modal knowledge in a single converged system. Memory is constructed, evolved, and retrieved directly where enterprise data lives, enabling Oracle to serve as the system of record for agents, memory. and enterprise data. Finally, we explain how native agent memory improves fundamental data operations such as NL2SQL. Sungpack Hong Sungpack Hong is Vice President of AI Research and Advanced Development, Database.
