SNU Data Science Seminar “Black-Box Model Alignment: Why It Matters and How We Can Approach It” Time : 09:00 - 10:30, Oct 15,2025 Zoom: https://snu-ac-kr.zoom.us/j/2507466555 Abstract This talk motivates post-hoc alignment for pretrained, black-box models, aiming for rigorous guarantees without retraining or touching weights. We first pose a general operating-point problem: maximize a primary objective (e.g., helpfulness) while keeping a guardrail metric (e.g., harmfulness) below a given target. We approach this problem using conformal risk control (CRC) to route between the outputs of two models (or a model and a human). We then show how CRC, paired with property- testing ideas, can check whether a model satisfies desired properties: and when it does not, we lift point predictions to set-valued outputs that, with high probability, contain a property-satisfying function. We will demonstrate applications in the medical domain, but emphasize the generality of these methods for providing auditable, model-agnostic control at the API layer.
